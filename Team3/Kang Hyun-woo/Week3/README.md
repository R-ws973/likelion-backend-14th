3주차 정기세션을 위한 PBL 과제를 안내드립니다.
이번 과제는 기초 → 조건문 → 반복문 흐름으로 구성되어 있습니다.

문제 1. 기본 계산 프로그램 (기초)
사용자로부터 두 개의 정수를 입력받아 기본 연산을 수행하는 프로그램을 작성하시오.

[요구사항]
1. 두 개의 정수를 입력받는다.
2. 다음 결과를 출력한다.
- 덧셈
- 뺄셈
- 곱셈
- 나눗셈 (정수 나눗셈)

[조건]
- Scanner를 사용하여 입력받을 것
- 출력 형식은 자유롭게 구성

  문제 2. 등급 판별 프로그램 (조건문)
  사용자로부터 점수를 입력받아 등급을 판별하는 프로그램을 작성하시오.

[요구사항]
1. 사용자로부터 점수(0~100)를 입력받는다.
2. 점수에 따라 등급을 출력한다.

[등급 기준]
- 90 이상 → A
- 80 이상 → B
- 70 이상 → C
- 그 외 → D

[조건]
- 조건문(if 또는 switch)을 반드시 사용할 것
- 잘못된 입력(0~100 범위 초과)에 대한 예외 처리 포함

  문제 3. 성적 입력 프로그램 (반복문)
  여러 학생의 점수를 입력받아 평균을 계산하는 프로그램을 작성하시오.

[요구사항]
1. 학생 수를 입력받는다.
2. 학생 수만큼 점수를 입력받는다.
3. 전체 평균을 출력한다.

[조건]
- 반복문(for 또는 while)을 반드시 사용할 것
- 배열을 활용하여 점수를 저장할 것
- 평균은 소수점까지 출력할 것
